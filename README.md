**openRS_** turns your phone or Android Auto head unit into a full telemetry dashboard for the Ford Focus RS MK3. Connects wirelessly to a [MeatPi WiCAN](https://www.wican.io/) and decodes 33 live parameters from the CAN bus and ECU — including data Ford never exposes.

| | |
|---|---|
| **6 screens** | DASH · AWD · PERF · TEMPS · TUNE · TPMS |
| **Hardware** | MeatPi WiCAN (Wi-Fi OBD-II) · Android 9+ |
| **Android Auto** | Custom UI — identical to phone app |
| **Theme** | Nitrous Blue `#00AEEF` · Frost White `#F5F6F4` |

→ **[Full README & build instructions](Android App/README.md)**
